# Ценообразование

## Формула
```
Финальная цена = базовая_цена × multiplier
multiplier = текущий_курс_USD / 80.62 (базовый курс)
```

## Welcome бонус
- Новый юзер: **300 кредитов** (fraud_score < 3)
- Подозрительный: **50 кредитов** (score 3-4)
- Заблокированный: **0** (score ≥ 5)

## Чат
- **1 кредит** за сообщение × multiplier
- Лимит: 50 сообщений в чате, 10000 символов
